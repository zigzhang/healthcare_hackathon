== README

Goal of this project: reduce errors input by nurses in Clinical studies and allow Sponsors to keep better track of those errors.

Both Android and Web App

Features:

* Generate QRCode by patient
* Scan QRCode
* Generate a Clinical Protocol
* Task Manager for nurse according to Patient and Protocol
* Analytics - Global stats per Sites + detect schedule errors in Protocol and Clinical trials
